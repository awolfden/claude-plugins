Submit Your Plugin

Share your Claude Code skills, hooks, and integrations with the community

1

Fork the Repository

Start by forking the Claude Plugins repository on GitHub:

git clone https://github.com/YOUR_USERNAME/claude-plugins.git
cd claude-plugins
2

Create Your Plugin File

Create a new JSON file in the plugins/ directory with your plugin metadata:

plugins/your-plugin-name.json

Use the template below as a starting point. The filename should match your plugin's ID.

3

Validate Your Plugin

Run the validation script to ensure your plugin meets the schema requirements:

npm run validate
4

Submit a Pull Request

Commit your changes and submit a pull request to the main repository. Include:

  • A clear description of what your plugin does
  • Any dependencies or requirements
  • Example usage (if applicable)
